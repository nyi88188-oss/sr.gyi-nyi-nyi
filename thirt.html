HTML Some Tags

    ၁။ Form (အချက်အလက် ကောက်ယူသည့် Tag များ)

  • User တွေဆီက အချက်အလက်တောင်းတဲ့အခါ (ဥပမာ - Login ဝင်တာ၊ အကောင့်သစ်ဖွင့်တာ၊ မက်ဆေ့ချ်ပိုတာ) တွေမှာ ဒီ Tag တွေကို အဓိက သုံးပါတယ်။
  • * < form>: ဖောင်တစ်ခုလုံးကို အုပ်ထားပေးတဲ့ အဓိက Tag ပါ။
  • * < input>: စာရိုက်ထည့်လိုရတဲ့ အကွက်လေးတွေ ဖန်တီးဖိုပါ။ အမျိုးအစား (Type) ပေါ်မူတည်ပြီး စာရိုက်ဖို၊ Password ထည့်ဖို၊ အမှန်ခြစ် (Checkbox) လုပ်ဖို စသဖြင့် ပြောင်းလဲသွားပါတယ်။
  • * < label>: အဲဒီ Input အကွက်တွေရဲ့ နာမည် (စာသား) ကို ဖော်ပြဖိုပါ။
  • * < button>: Click နှိပ်လိုရတဲ့ ခလုတ်လေးတွေ ဖန်တီးဖိုပါ။ (ဥပမာ- "Submit" သိုမဟုတ် "Login" ခလုတ်)
  • * < textarea>: စာအရှည်ကြီးတွေ (ဥပမာ- Comment သိုမဟုတ် Message) ရိုက်ထည့်ဖို နေရာကျယ်ကျယ် ဖန်တီးပေးပါတယ်။

    • ၂။ Text Formatting (စာသား ပုံစံပြောင်း Tag များ) စာသားတွေကို ပိုပြီး ပေါ်လွင်အောင် အလှဆင်တဲ့အခါ သုံးပါတယ်။

  • < i> သိုမဟုတ် < em>: စာလုံးစောင်း (Italic) ဖြစ်အောင် လုပ်ပေးပါတယ်။
  • < b>: စာလုံးအမည်း/အထူ (Bold) ဖြစ်အောင် လုပ်ပေးပါတယ်။ (< strong> နဲ့ အလုပ်လုပ်ပုံ ဆင်တူပါတယ်)
  • < mark>: စာသားကို Highlighter တားထားသလို အဝါရောင်လေးနဲ့ မှတ်ထားချင်ရင် သုံးပါတယ်။
  • < blockquote>: သူများပြောခဲ့တဲ့ စာသား ဒါမှမဟုတ် ကိုးကားချက် (Quote) တွေကို သီးသန့် ဖယ်ထုတ်ပြချင်ရင် သုံးပါတယ်။

    • ၃။ Table (ဇယားကွက် အစိတ်အပိုင်း Tag များ) အရင်က < table> ဆိုတဲ့ Tag အကြောင်း ပြောခဲ့ပေမဲ့ အဲဒီ ဇယားထဲမှာ အချက်အလက်ထည့်ဖို အောက်ပါ Tag လေးတွေ ထပ်ပြီး တွဲသုံးရပါတယ်။

  • < tr> (Table Row): ဇယားရဲ့ အတန်းတစ်တန်းကို ဖန်တီးပါတယ်။
  • < th> (Table Header): ဇယားရဲ့ ခေါင်းစဉ် (Column Name) အတွက် သုံးပါတယ်။ စာလုံးနည်းနည်း ပိုထူပြီး အလယ်မှာ ပေါ်လေ့ရှိပါတယ်။
  • < td> (Table Data): ဇယားထဲက အချက်အလက် အကွက်တစ်ခုချင်းစီ ထည့်ဖို သုံးပါတယ်။

    • အခြားအရေးကြီးသော Tag များ

  • < iframe>: ကိုယ့် Website ထဲမှာ တခြား Website ရဲ့ အစိတ်အပိုင်း (ဥပမာ- YouTube Video, Google Maps) တွေကို မြှုပ်ပြီး ထည့်ချင်ရင် သုံးပါတယ်။

    • ယခုက အခြေခံလေးများကိုပြောပြ ပေးသွားတာပါ။ နောင်တစ်ခါမှာတော့ အခြေခံအကြောင်းတွေကို နားလည်လောက်မယ်လို့ထင်ပါတယ်။ လိုအပ်ချက်များရှိသွာရင်း ပိုမိုကောင်းမွန်သောပြင်ဆင်မှုများနှင့် ပြန်လာပါမယ်။
