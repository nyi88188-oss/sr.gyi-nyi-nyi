• HTML ဆိုတာကတော့ HyperText Markup Language ရဲ့ အတိုကောက်ဖြစ်ပြီး Website တစ်ခုရဲ့ တည်ဆောက်ပုံ (Structure) ကို ဖန်တီးတဲ့နေရာမှာ အခြေခံအကျဆုံး ဘာသာစကားတစ်ခု ဖြစ်ပါတယ်။
  • ရိုးရိုးရှင်းရှင်း ပြောရရင် HTML ဟာ Website တစ်ခုရဲ့ "အရိုးစု" ဒါမှမဟုတ် "အုတ်မြစ်" နဲ့ တူပါတယ်။
  • HTML က ဘာတွေလုပ်ပေးတာလဲ?
  • HTML ဟာ Programming Language တစ်ခု မဟုတ်ဘဲ Markup Language တစ်ခု ဖြစ်ပါတယ်။ သူက Browser (Chrome, Firefox စတာတွေ) ကို Website ရဲ့ Content တွေဖြစ်တဲ့ စာသား၊ ပုံ၊ ဗီဒီယိုတွေကို ဘယ်လိုပြသရမလဲဆိုတာ ညွှန်ကြားပေးပါတယ်။
    • Heading တွေ သတ်မှတ်ပေးခြင်း: ခေါင်းစဉ်ကြီး၊ ခေါင်းစဉ်သေး ခွဲခြားပေးပါတယ်။ Paragraph တွေ ဖန်တီးခြင်း: စာပိုဒ်တွေကို စနစ်တကျ စီပေးပါတယ်။ Links များ ချိတ်ဆက်ခြင်း: တစ်မျက်နှာနဲ့ တစ်မျက်နှာ ချိတ်ဆက်ပေးပါတယ်။ Images & Media: ပုံတွေနဲ့ ဗီဒီယိုတွေကို ထည့်သွင်းပေးပါတယ်။
  • HTML ရဲ့ အဓိက အစိတ်အပိုင်းများ (Tags) < >
    • HTML ကို Tags လို့ခေါ်တဲ့ အမှတ်အသားတွေနဲ့ ရေးသားပါတယ်။ များသောအားဖြင့် Tag တစ်ခုမှာ အဖွင့် (<->) နဲ့ အပိတ် (< /->) ဆိုပြီး ရှိတတ်ပါတယ်။



    ၁။ မရှိမဖြစ် အခြေခံ Tag များ (Essential Structure)
  • Website တစ်ခုမှာ ဒီ Tag တွေ မပါလို့ မရပါဘူး။
  • < html >: တစ်ခုလုံးရဲ့ အစနဲ့ အဆုံးကို သတ်မှတ်ပါတယ်။
  • < head>: Website ရဲ့ အချက်အလက်တွေ (ဥပမာ- Title, CSS link, Favicon) ထည့်တဲ့နေရာဖြစ်ပြီး Browser ထဲမှာ တိုက်ရိုက်မမြင်ရပါဘူး။
  • < title>: Browser ရဲ့ Tab မှာ ပြမယ့် Website နာမည်ကို ရေးပါတယ်။
  • < body>: လူတွေမြင်ရမယ့် Content အားလုံး (စာ၊ ပုံ၊ ဗီဒီယို) ကို ဒီထဲမှာပဲ ရေးရပါတယ်။



    • ၂။ စာသားဆိုင်ရာ Tag များ (Text Content)
    • စာသားတွေကို ပုံစံအမျိုးမျိုးနဲ့ ပြသဖို သုံးပါတယ်။
    • < h1 > မှ < h6 >: ခေါင်းစဉ် (Heading) များအတွက် သုံးပါတယ်။ < h1 > က အကြီးဆုံးဖြစ်ပြီး < h6 > က အသေးဆုံးပါ။
    • < p >: စာပိုဒ် (Paragraph) ရေးဖို သုံးပါတယ်။
    • < span >: စာကြောင်းတစ်ကြောင်းတည်းက စာသားအချိုကိုပဲ သီးသန့်အရောင်ခြယ်တာမျိုး လုပ်ချင်ရင် သုံးပါတယ်။
    • < br >: စာကြောင်းအသစ် (Line break) ဆင်းချင်ရင် သုံးပါတယ်။
    • < strong >: စာသားကို အထူးပြု (စာလုံးအထူ) လုပ်ချင်ရင် သုံးပါတယ်။
    • < marquee>: ကစာသားကိုရွှေ့လျားစေသည့်နေရာတွင် အသုံးပြုပါတယ်။



    ၃။ ချိတ်ဆက်မှုနှင့် မီဒီယာ Tag များ (Links & Media)Website ကို အသက်ဝင်စေတဲ့ အစိတ်အပိုင်းတွေပါ။
    • < a >: တစ်ခြား Page တွေကို သွားဖို Link ချိတ်ပေးပါတယ်။ (ဥပမာ- < a href="url">စာသား< /a >)
    • < img >: ဓာတ်ပုံတွေ ပြသဖို သုံးပါတယ်။ (ဥပမာ-_< img src="မိမိထည့်မည့်ပုံလင့်" alt="ဘာပုံဖြစ်ကြောင်းထည့်" height="လိုချင်တဲ့ဆိုဒ်ကို ရေးပြီး(px) ထည့်" >)
    • < video > / < audio >: လည်းအတူတူပါဘဲ။ သူက ဗီဒီယိုနဲ့ အသံဖိုင်တွေ ထည့်သွင်းဖို သုံးပါတယ်။
